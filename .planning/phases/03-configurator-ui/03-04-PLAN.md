    ```typescript
    $effect(() => {
        // Track the derived value to create dependency
        const _snapshot = contentSnapshot;

        if (isInitialLoad) {
            isInitialLoad = false;
            return;
        }

        if (saveTimeoutId) clearTimeout(saveTimeoutId);
        saveStatus = 'saving';

        saveTimeoutId = setTimeout(async () => {
            // Read current content inside setTimeout to avoid stale closure (pitfall 3)
            if (diagramStore.currentDiagram) {
                await diagramStore.updateContent(service, diagramStore.currentDiagram.content);
                if (!diagramStore.error) {
                    saveStatus = 'saved';
                    // Generate and persist thumbnail after successful save
                    await generateAndPersistThumbnail();
                } else {
                    saveStatus = 'error';
                    // Toast for errors only (D-15)
                    toast.error('Changes could not be saved. Check your connection and try again.');
                }
            }
        }, 500);

        return () => {
            if (saveTimeoutId) clearTimeout(saveTimeoutId);
        };
    });
    ```

    **Thumbnail generation function (D-16):**
    ```typescript
    async function generateAndPersistThumbnail(): Promise<void> {
        if (!diagramStore.currentDiagram) return;

        try {
            const svgElement = document.querySelector('svg[role="img"]');
            if (!svgElement) return;

            const svgData = new XMLSerializer().serializeToString(svgElement);
            const blob = new Blob([svgData], { type: 'image/svg+xml;charset=utf-8' });
            const url = URL.createObjectURL(blob);

            const thumbnail = await new Promise<string | null>((resolve) => {
                const img = new Image();
                img.onload = () => {
                    const canvas = document.createElement('canvas');
                    canvas.width = 300;
                    canvas.height = 200;
                    const ctx = canvas.getContext('2d')!;
                    ctx.drawImage(img, 0, 0, 300, 200);
                    URL.revokeObjectURL(url);
                    resolve(canvas.toDataURL('image/png', 0.8));
                };
                img.onerror = () => {
                    URL.revokeObjectURL(url);
                    resolve(null);
                };
                img.src = url;
            });

            if (thumbnail) {
                // Persist thumbnail via a separate update call (not updateContent which replaces full content)
                await service.update(diagramStore.currentDiagram.id, { thumbnail_base64: thumbnail });
            }
        } catch {
            // Thumbnail generation is best-effort -- don't fail the save
            console.warn('Thumbnail generation failed');
        }
    }
    ```

    **Important implementation notes:**
    - Import `toast` from `svelte-sonner` (for error toasts only per D-15)
    - The `isInitialLoad` flag prevents the first `$effect` run from triggering a save (pitfall 1 in RESEARCH.md)
    - Read `diagramStore.currentDiagram.content` inside the setTimeout callback to get latest value (pitfall 3 in RESEARCH.md)
    - Thumbnail canvas size: 300x200 pixels, PNG quality 0.8 (per RESEARCH.md pattern)
    - SVG uses inline styles (REND-03) so no external stylesheet issues with canvas serialization
    - All logos are data: URIs from logo proxy (Phase 2), so no canvas taint risk (pitfall 2 in RESEARCH.md)
    - The thumbnail is persisted via `service.update()` with just `{ thumbnail_base64 }` -- this is a separate PATCH call from the content save, keeping the auto-save lightweight
