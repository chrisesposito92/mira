```svelte
<svelte:options namespace="svg" />

<script lang="ts">
  import { PILL_PADDING_X, PILL_PADDING_Y, PILL_BORDER_RADIUS, PILL_TEXT, SVG_FONT_FAMILY } from '../constants.js';
  import { estimatePillWidth } from '$lib/utils/diagram-layout.js';

  let { x, y, label, color }: {
    x: number;
    y: number;
    label: string;
    color: string;
  } = $props();

  const fontSize = 10;
  const pillWidth = $derived(estimatePillWidth(label, fontSize));
  const pillHeight = $derived(fontSize + PILL_PADDING_Y * 2);
</script>

<g>
  <rect
    x={x - pillWidth / 2}
    y={y - pillHeight / 2}
    width={pillWidth}
    height={pillHeight}
    rx={PILL_BORDER_RADIUS}
    style="fill: {color};"
  />
  <text
    {x}
    y={y}
    text-anchor="middle"
    dominant-baseline="central"
    style="font-family: {SVG_FONT_FAMILY}; font-size: {fontSize}px; font-weight: 500; fill: {PILL_TEXT};"
  >
    {label}
  </text>
</g>
```

The component must:
1. Look up source and target node positions from the layout. Search through `layout.prospect`, `layout.groups[].systems[]`, `layout.standalone[]`, and synthesize hub position from `layout.hub`. Match by `connection.source_id` and `connection.target_id` against system IDs or special IDs like `'hub'` for the m3ter hub node.
2. Compute source center (sx, sy) and target center (tx, ty) from the positioned node x/y/width/height.
3. Get color from `CONNECTION_COLORS[connection.connection_type]` with fallback to `CONNECTION_COLORS.api` (#90A4AE).
4. Render `<line>` with:
   - `x1={sx} y1={sy} x2={tx} y2={ty}`
   - `style="stroke: {color}; stroke-width: {CONNECTION_STROKE_WIDTH}; stroke-dasharray: {CONNECTION_DASH};"`
   - For unidirectional: `marker-start="url(#source-dot)"` and `marker-end="url(#arrowhead)"`
   - For bidirectional: `marker-start="url(#arrowhead)"` and `marker-end="url(#arrowhead)"`
5. Compute midpoint using `getConnectionMidpoint(sx, sy, tx, ty)`.
6. Render `<ConnectionPill x={mid.x} y={mid.y} {label} {color} />` if `connection.label` is non-empty and the diagram settings `show_labels` is true (passed via additional prop or always shown per UI-SPEC).

Create `connections/` directory. All SVG elements use inline `style` attributes -- no `class=` attributes.
