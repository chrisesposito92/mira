  import { PILL_PADDING_X, PILL_PADDING_Y, PILL_BORDER_RADIUS, PILL_TEXT, SVG_FONT_FAMILY, MAX_PILL_LABEL_CHARS } from '../constants.js';

  let { x, y, label, color }: {

  } = $props();

  const fontSize = 10;
  const truncatedLabel = $derived(truncateSvgText(label, MAX_PILL_LABEL_CHARS));
  const pillWidth = $derived(estimatePillWidth(truncatedLabel, fontSize));
  const pillHeight = $derived(fontSize + PILL_PADDING_Y * 2);
</script>

  <rect
    x={x - pillWidth / 2}
    y={y - pillHeight / 2}
    width={pillWidth}
    height={pillHeight}
    rx={PILL_BORDER_RADIUS}
    style="fill: {color};"

    {x}
    y={y}
    text-anchor="middle"

**Addresses review MEDIUM-HIGH: Lines use edge-anchor calculation, not center-to-center.**
**Addresses review MEDIUM: show_labels passed explicitly, not ambiguous.**

Props:
```typescript
{
  connection: DiagramConnection;
  nodePositions: NodePositionMap;
  showLabels: boolean;
}
```

The component must:
1. Look up source and target bounding boxes from `nodePositions[connection.source_id]` and `nodePositions[connection.target_id]`. If either is missing, render nothing (skip gracefully).
2. Compute source center (sCx, sCy) and target center (tCx, tCy) from the node bounding boxes: `sCx = sourceRect.x + sourceRect.width / 2; sCy = sourceRect.y + sourceRect.height / 2`.
3. Use `computeEdgeAnchor(sourceRect, tCx, tCy)` to get the source edge point (sx, sy).
4. Use `computeEdgeAnchor(targetRect, sCx, sCy)` to get the target edge point (tx, ty).
5. Get color from `CONNECTION_COLORS[connection.connection_type]` with fallback to `CONNECTION_COLORS.api` (#90A4AE).
6. Render `<line>` with:
   - `x1={sx} y1={sy} x2={tx} y2={ty}`
   - `style="stroke: {color}; stroke-width: {CONNECTION_STROKE_WIDTH}; stroke-dasharray: {CONNECTION_DASH};"`
   - For unidirectional: `marker-start="url(#source-dot)"` and `marker-end="url(#arrowhead)"`
   - For bidirectional: `marker-start="url(#arrowhead)"` and `marker-end="url(#arrowhead)"`
7. Compute midpoint using `getConnectionMidpoint(sx, sy, tx, ty)`.
8. If `showLabels` is true AND `connection.label` is non-empty: render `<ConnectionPill x={mid.x} y={mid.y} label={connection.label} {color} />`.

Create `connections/` directory. All SVG elements use inline `style` attributes -- no `class=` attributes.

Create test file. Tests render ConnectionLine inside a wrapping `<svg>` element. Create a minimal mock NodePositionMap with two positioned nodes for lookups. Assert on rendered attributes.
